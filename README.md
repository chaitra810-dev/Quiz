# 🧠 SkillCheck Pro – Technical Knowledge Assessment Portal

 📌 About the Project

SkillCheck Pro is a responsive Technical Knowledge Assessment Portal developed using HTML, CSS, and JavaScript. The application is designed to provide an engaging and interactive quiz experience where users can evaluate their programming and web development knowledge through multiple-choice questions.

The project demonstrates JavaScript concepts such as DOM manipulation, event handling, score calculation, dynamic question rendering, and responsive interface design while providing a clean and professional user experience.
